Data Visualizations of COVID-19 Vaccinations, Cases, and Deaths in Illinois¶

J Montgomery Maxwell¶

07/15/2021

This notebook is a snapshot of publically available data from IDPH related to COVID-19 in Illinois. Data visualizations illustrate the total and daily administed vaccine doses, the total of fully vaccinated people in Illinois, and the percentage of the population of Illinois who have received at least one vaccine dose. There are additional graphics for the daily changes in the number of confirmed cases and the number of COVID-19 related deaths.

This data is in no way exhaustive and the analysis and graphics presented here should not be considered as part of a rigorous study of COVID-19 or of public health policy. The purpose of this notebook is merely to demonstrate the advantages and capabilities of using the University of Chicago's Center for Translational Data Science's Pandemic Data Commons.

Import Data¶

Import historic data of Illinois's vaccinations, cases, tests, and deaths relating to COVID-19.

  • Data Source: IDPH
  • Cases, Tests & Deaths API: https://idph.illinois.gov/DPHPublicInformation/api/COVIDExport/GetIllinoisCases
  • Vaccinations API: https://idph.illinois.gov/DPHPublicInformation/api/COVIDExport/GetVaccineAdministration?countyname=Illinois

cases_url = (
    "https://idph.illinois.gov/DPHPublicInformation/api/COVIDExport/GetIllinoisCases"
)
request = urllib.request.urlopen(cases_url)
json_data = json.loads(request.read().decode())
cases = pd.json_normalize(json_data)

vaccinations_url = "https://idph.illinois.gov/DPHPublicInformation/api/COVIDExport/GetVaccineAdministration?countyname=Illinois"
request = urllib.request.urlopen(vaccinations_url)
json_data = json.loads(request.read().decode())
vaccinations = pd.json_normalize(json_data)

Clean the data for analysis and visualization. Create an additional dataframe for cases, tests, and deaths that matches the indexed dates for the historic vaccination data.

def thousands(x, pos):
    return "%1.0fK" % (x * 1e-3)


def millions(x, pos):
    return "%1.0fM" % (x * 1e-6)


def percents(x, pos):
    y = int(x * 100)
    return f"{y}%"


asofYr = vaccinations.index[-1:].values[0][2:4]
asofMo = vaccinations.index[-1:].values[0][5:7]
asofDay = vaccinations.index[-1:].values[0][8:10]

fig, ax = plt.subplots(figsize=(9, 6), dpi=100)

Summary¶

We can see from the first graphic that the number of daily administered vaccinations grew steadily until mid April when a sharp decline occured in the number of new doses administered. Note that this decline in daily new vaccinations corresponds with the suspension and re-approval of the Johnson & Johnson vaccine. From the second graphic of the rolling average in daily administered vaccines and new fully vaccinated people we can see that the decline in administered vaccines closely parallels the decline in new fully vaccinated people. Since the Johnson & Johnson vaccine is a single dose vaccine and a suspension of this vaccine would have an immediate effect on the number of new fully vaccinated people, the second graphic further suggests that these declines in administered vaccines and new fully vaccinated people corresponds with the suspension of the Johnson & Johnson vaccine. From the third graphic of total vaccine recipients it is clear that the number of fully vaccinated people is growing along with the total vaccine doses administered, this suggests that vaccine recipients are returning for their second dose.

In the fourth graphic, relating the daily confirmed new cases with the percentage of Illinois residents who have been fully vaccinated we can see that despite the growth in the percentage of vaccinated residents, there was a temporary increase in the number of daily new cases. This spike in cases could be for a variety of reasons such as changes in weather or public policy regarding indoor spaces which would lead to an increase in mobility, or from the introduction of a virus variant from a different country.

Looking at the fifth graphic, a comparison of daily reported deaths related to COVID-19 and the percentage of fully vaccinated residents we can see a dramatic decrease in the number of daily COVID-19 related deaths. However, this comparison between the vaccination percentages and the number of reported deaths is limited, we must investigate further the relation between the number of fully vaccinated people, the number of new cases, and the daily reported deaths.
